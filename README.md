## 📝 Personnalisation

### Couleurs
Modifiez les variables CSS dans `style.css`:
```css
:root {
    --first-color: #6366f1;
    --second-color: #0ea5e9;
    --accent-color: #ec4899;
}
```

### Animations
Ajustez la vitesse des animations dans `animations.css`:
```css
.animate-fade-in {
    animation-duration: 0.8s; /* Modifier ici */
}
```
